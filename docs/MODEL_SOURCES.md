## Alternative: Generate Procedurally

For asteroids and smaller bodies, you can use procedural generation in Blender:
1. Create UV sphere
2. Apply Subdivision Surface modifier
3. Add Displacement modifier with Noise texture
4. Apply rocky/metallic materials
5. Export as .glb with Draco compression
